# Todo-app

A modern, responsive, and installable Task Management Web Application built using HTML, CSS, and JavaScript. Designed to help users organize tasks efficiently with categories, priorities, due dates, search, sorting, filtering, progress tracking, dark mode, and offline support.

## Features

### Task Management

* Add new tasks instantly
* Assign task categories
* Set task priorities
* Add due dates
* Mark tasks as completed
* Delete tasks

### Categories

Organize tasks into:

* Work
* Personal
* Study

### Priority Levels

* High Priority
* Medium Priority
* Low Priority

### Search Functionality

* Search tasks in real-time
* Quickly find specific tasks

### Progress Tracking

* Visual progress bar
* Track completed vs pending tasks

### Filtering Options

* All Tasks
* Completed Tasks
* Pending Tasks

### Due Date Management

* Set deadlines for tasks
* Keep track of upcoming work

### Sorting Options

* Newest First
* Oldest First
* Priority Based

### Dark Mode

* One-click dark/light theme switching
* Comfortable viewing experience

### Progressive Web App (PWA)

* Installable on mobile and desktop
* Works offline
* App-like experience

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Progressive Web App (PWA)
* Local Storage

## Installation

## Install as an App

### Desktop

1. Open the website in Chrome or Edge.
2. Click the Install App icon in the address bar.
3. Confirm installation.

### Mobile

1. Open the website.
2. Tap "Add to Home Screen".
3. Install the application.
